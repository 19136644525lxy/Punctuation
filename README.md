# Punctuation - 标点模组

一个 Minecraft Fabric 模组，允许玩家使用鼠标中键标记位置并进行传送。

## 功能特性

- **标点功能**：使用鼠标中键标记任意方块位置
- **传送功能**：按 P 键传送至标记位置
- **模式切换**：按左 Alt 键在标点模式和选取方块模式间切换
- **视觉效果**：标点位置显示高亮边框和自定义图标
- **配置界面**：支持自定义边框颜色方案（14种颜色可选）
- **支持渐变**：边框颜色支持渐变色效果

## 操作说明

| 按键 | 功能 |
|------|------|
| 鼠标中键 | 在标点模式下标记位置 |
| P | 传送至标记位置 |
| 左 Alt | 切换标点模式/选取方块模式 |

## 配置

通过 ModMenu 打开配置界面，可以设置：
- 是否显示高亮边框
- 边框颜色方案
- 是否使用渐变色

## 前置依赖

- Fabric API
- Cloth Config API
- ModMenu

## 支持版本

- Minecraft 1.20.1

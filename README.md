# Punctuation - 标点模组

一个 Minecraft Fabric 模组，允许玩家使用鼠标中键标记位置并进行传送，同时提供 API 支持附属模组扩展功能。

## 功能特性

### 核心功能
- **标点功能**：使用鼠标中键标记任意方块位置
- **传送功能**：按 P 键传送至标记位置
- **模式切换**：按左 Alt 键在标点模式和选取方块模式间切换
- **视觉效果**：标点位置显示高亮边框和自定义图标
- **配置界面**：支持自定义边框颜色方案（14种颜色可选）
- **支持渐变**：边框颜色支持渐变色效果

### API 功能
- **MarkerEvent**：提供标点事件回调，支持附属模组监听标点行为
- **标记信息查询**：获取标记位置、实体ID、实体位置、标记类型、物品栈等信息
- **网络同步**：支持客户端-服务端标记数据同步

## 操作说明

| 按键 | 功能 |
|------|------|
| 鼠标中键 | 在标点模式下标记位置 |
| P | 传送至标记位置 |
| 左 Alt | 切换标点模式/选取方块模式 |
| Y | 同意传送请求 |
| N | 取消传送请求 |

## 配置

通过 ModMenu 打开配置界面，可以设置：
- 是否显示高亮边框
- 边框颜色方案
- 是否使用渐变色
- 标记广播持续时间（1-10秒）

## 附属模组

### Alone Host (ah)

附属模组，利用主模组的标点功能实现敌对生物友好化和控制。

#### 功能特性
- **生物友好化**：使用生物盒右键敌对生物使其友好化
- **生物收纳**：右键已友好化的生物将其收纳到生物盒中（容量10只）
- **生物释放**：Shift+右键方块释放生物盒中的生物
- **AI行为模式**：
  - 待机模式：等待指令（默认）
  - 跟随模式：跟随玩家，64格内攻击敌对生物，超距自动传送
  - 巡逻模式：前往标点位置巡逻，64格内攻击敌对生物
- **行为切换**：空手右键友好化生物循环切换行为模式
- **增益效果**：友好化生物获得1000血量、力量4、抗性提升，亡灵生物额外获得力量6
- **亡灵保护**：友好化后的亡灵生物不会被太阳灼烧

#### 操作说明
| 操作 | 功能 |
|------|------|
| 生物盒右键敌对生物 | 使其友好化 |
| 生物盒右键友好化生物 | 收纳到生物盒 |
| Shift+生物盒右键方块 | 释放生物盒中的生物 |
| 空手右键友好化生物 | 切换行为模式 |
